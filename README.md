# 🌿 Green Buddy

**Green Buddy** เป็นแอปพลิเคชัน Flutter ที่ช่วยให้คุณดูแลสวนของตัวเองอย่างมีประสิทธิภาพ ติดตามการเจริญเติบโตของพืช รับเตือนการรดน้ำ และเก็บเอกสารสภาพของแต่ละพืช

## ✨ ฟีเจอร์หลัก

- 📱 **ติดตามพืชแบบเรียลไทม์** - ดูแลพืชทั้งหมดในแอปพลิเคชันเดียว
- 💧 **การเตือนการรดน้ำ** - รับการแจ้งเตือนอัตโนมัติตามตารางการดูแล
- 📸 **อัพโหลดรูปภาพ** - บันทึกภาพของพืชเพื่อติดตามการเจริญเติบโต
- 🔥 **Firebase Integration** - คลาวด์บันทึกข้อมูลอย่างปลอดภัย
- 📊 **สถิติและรายงาน** - ดูข้อมูลการเจริญเติบโตและประวัติการดูแล
- 🌍 **มัลติแพลตฟอร์ม** - ใช้ได้บน Android, iOS, Web, Windows, macOS และ Linux

## 🛠️ เทคโนโลยีที่ใช้

- **Framework**: Flutter 3.11.0+
- **Language**: Dart
- **Backend**: Firebase
- **Database**: Cloud Firestore
- **Authentication**: Firebase Authentication
- **Storage**: Firebase Storage

## 📦 Dependencies

- firebase_core: ^4.4.0
- cloud_firestore: ^6.1.2
- firebase_auth: ^6.1.4
- image_picker: ^1.2.1
- firebase_storage: ^13.0.6
- intl: ^0.20.2
